# alicelinux
An experimental minimal musl based distro

## Host Build Dependencies
* rsync
* git
* curl
* base-devel (gcc)
* ccache

### List for Build
1. Execute ./genconf generate xpkg.conf for build-cross 
2. Execute ./build for build cross-compile
